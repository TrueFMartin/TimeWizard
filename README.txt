run buildWindows.bat to compile and run on windows system
run buildLinuxMax.sh to compile and run on linux/mac
    may have to edit bash path in first line of .sh file
    may have to run bash as sudo

source .java are in ./src/main
resources are in ./src/bin/resources
class files will write to ./src/bin/main

Requires enhanced switch statment, update java to 13+ if run fails
